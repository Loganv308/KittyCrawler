# KittyCrawler

KittyCrawler is a Java-based application designed to scrape data from a local cat breeders’ website, monitoring for new entries. The project was developed as a hands-on exercise to deepen proficiency in Java, with a focus on key areas such as web scraping, object-oriented programming, algorithm design, and writing production-ready code.

This program has already proven effective in practice, successfully notifying me of new entries on two occasions. As a direct result, we were able to adopt two cats by securing an early opportunity—demonstrating the program’s real-world usefulness beyond its development goals.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Loganv308/KittyCrawler.git
```

Go to the project directory

```bash
  cd KittyCrawler
```

Compile the JAR file (Located in target folder)

```bash
  mvn compile
```
